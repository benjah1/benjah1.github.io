3D Terminology

  • Vertex
    A 3-dimensional coordinate, which in groups comprise a polygon.
    Edge
    A wire-like line representing the boundary of 2 adjacent vertices.
    Face
    A flat connection of edges representing a boundary, field, or solid surface.
  • Polygonal Model
    Triangle
    Quadrilateral
    Texture
    Textures are two dimensional image files that can be mapped onto the model's 3D surface through a process known as texture mapping.
    Shader
    A shader is a set of instructions applied to a 3D model that lets the computer know how it should be displayed.
  • * Also, Light source and Reflection
